# NodeJS

## 把项目交给NodeJS管理，会自动生成默认的package.json

- `npm init -y` 

## 安装依赖

- `npm install`  依赖的名字
  - 例 `npm install `

## 写入到 dependencies 对象 （生产依赖）

- `npm i module_name  -S  = >  npm install module_name --save`

## 写入到 devDependencies 对象 （开发依赖）

- `npm i module_name  -D  => npm install module_name --save-dev`

## 全局安装

- `npm i module_name  -g`
